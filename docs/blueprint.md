# **App Name**: LexiGen

## Core Features:

- Article Definition and History: Provides a brief definition and historical context for legal articles using the Gemini API. Gemini acts as a tool, reasoning whether a definition and history are relevant and appropriate to generate.
- Homepage Article Display: Displays articles and legal news in a continuously scrolling format on the homepage.
- LawGPT Integration: A button that navigates the user to the LawGPT chat interface.
- Chat History: Saves the history of chats within the LawGPT interface in a database.
- Authentication and Login: Provides secure user authentication and login functionality.
- Emergency Numbers Display: Displays a list of emergency contact numbers at the top of the application.
- Connect with Lawyers Link: Provides a link for users to connect with available lawyers.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and authority.
- Background color: Light gray (#ECEFF1) for a clean and professional look.
- Accent color: Muted orange (#FF9800) for highlights and call-to-action buttons.
- Body and headline font: 'Inter', a grotesque sans-serif font with a modern, objective, neutral look, is suitable for both headlines and body text.
- Use professional, clear icons to represent different legal categories and functions.
- Maintain a clean and organized layout, with clear navigation and a focus on readability.
- Subtle transitions and animations to enhance user experience without being distracting.
